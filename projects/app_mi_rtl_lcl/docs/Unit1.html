Descrição

A unit Unit1 é usado para demonstração do uso da unit da mi_ui_mi_msgbox_dm.

  • NOTAS

    • A unit mi_ui_mi_msgbox_dm não precisa ser inicializada no projeto principal porque a mesma auto se cria caso ela seja declarada no projeto principal.

    • Os dialogos da LCL são executados indiretamente pela classe TMI_MsgBox.

      • Motivo:

        • A classe TMI_MsgBox pode ser usada em todos os tipos de aplicação, inclusive javascript, pelo menos essa é a ideia.

    • O projeto mi.rtl.tests tem um exemplo completo de todas as funcionalidades do pacote mi.rtl
