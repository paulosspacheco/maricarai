Descrição

OpenSSL support

This unit is Pascal interface to OpenSSL library (used by ssl_openssl unit). OpenSSL 1.1 is loaded dynamicly on-demand. If this library is not found in system, requested OpenSSL function just return errorcode.

Uses

  • Classes
  • synafpc
  • Libc
  • SysUtils
