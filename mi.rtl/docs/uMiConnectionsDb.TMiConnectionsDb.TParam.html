uMiConnectionsDb

Declaração

Descrição

O registro TParam é usado para transferir os parâmetros entre os métodos do DataModule dm_connections.

Descrição

Campos

ConnectorType:string;

O campo ConnectorType informa ao atributo SQLConnector1 o tipo de banco de dados que será usado.

  • Nota

    • Fiz testes com o banco de dados PostgresSQL

HostName:string;

O campo HostName informa o endereço IP do banco de dados.

UserName:string;

O campo UserName informa o nome do usuário administrador do banco de dados.

Password:string;

O campo Password informa a senha do usuário administrador do banco de dados.
