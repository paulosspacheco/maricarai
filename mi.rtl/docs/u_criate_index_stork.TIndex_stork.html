type TIndex_stork = class(TPageProducer)

Descrição

A classe TIndex_stork tem como objetivo criar o arquivo de configuração index_stork.toml que programa rust stork usa para crie o meta-dados index_stork.st para o programa javascript stork.js.

  • Template usado:

    • Nome do arquivo de templates:

      • './templates/stork/index_stork.toml'

base_directory = ""
url_prefix = ""

function CrateIndexStork:Integer;

O método CrateIndexStork salva a propriedade HTMLContent no arquivo './index_stork.torm'

  • Objetivo:

    • O arquivo './index_stork.torm' é usado pelo rograma rust stork para criar o arquivo list_stork.st usado pelo programa stork.js do site.

  • Após execução deste método, executar o comando abaixo no prompt do sistema operacional. 

    
# Cria índice para stork.js
stork build --input index_stork.toml --output list_stork.st
