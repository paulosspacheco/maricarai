Declaração

type TDM_Connections = class(TDataModule)

Descrição

O Data Module TDM_Connections é usado para conectar-se ao banco de dados através de parâmetros salvos no arquivo config.ini

procedure DataModuleCreate(Sender: TObject);

O método DataModuleCreate checa se o arquivo config.ini existe, se existir, ler os parãmetros de conexão do arquivo config.ini, se não existir pede para a pessoa que executou o programa informar parâmetros para conexão e em seguida salva no arquivo config.ini.

  • PARÂMETROS DE CONEXÃO

    • SESSÃO CONEXÃO;

      • ConnectorType;

      • UserName;

      • Password;

      • HostName;
